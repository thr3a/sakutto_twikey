## How To Use

Install

```
git clone https://github.com/thr3a/sakutto_twikey
cd sakutto_twikey
bundle install
```

Config

```
export TWITTER_KEY=AAAAAAA
export TWITTER_SECRET=AAAAAAA
```

Run

```
rails s
```

Access `http://localhost:3000/`
